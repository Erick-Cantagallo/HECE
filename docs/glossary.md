# Glossary

- **Axiom**: A fundamental scientific truth (e.g., Laws of Thermodynamics) injected by the Knowledge Engine.
- **Constraint**: A limitation defined by the user or physical reality that a hypothesis cannot violate.
- **Feasibility Score**: A value (0.0 to 1.0) representing the likelihood of a hypothesis being valid based on current evidence.
- **Hypothesis**: A structured scientific proposition generated by the Inference Engine.
- **Inference Engine**: The module responsible for AI-driven hypothesis generation.
